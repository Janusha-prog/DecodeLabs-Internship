# Rule-Based AI Chatbot  This is my first AI internship project developed using Python. 

This project is a simple "Rule-Based AI Chatbot" developed using "Python" as part of my first AI internship project at "Decode Labs".
The chatbot responds to predefined user inputs using a dictionary-based approach and demonstrates the basic working of conversational AI systems.

## 📌 Features

- 👋 Greeting responses
- 💬 Dictionary-based chatbot system
- 🧹 Input sanitization
- 🔁 Infinite conversation loop
- ❌ Exit command support
- ⚡ Fast text-based interaction
- 
## 🛠️ Technologies Used
- Python

## 🚀 How It Works

1. User enters a message
2. Chatbot checks the predefined responses
3. Matching response is displayed
4. Conversation continues until user exits
